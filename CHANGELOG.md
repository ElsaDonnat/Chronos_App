# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-02-23

### Added
- Complete Chronos history learning app
- 60 historical events across 10 progressive lessons (Chapter 1: The Story of Humanity)
- Learn tab with lesson list, event cards, and three-question quiz flow (Where/When/What)
- Timeline tab with vertical chronological explorer, era bands, and category filters
- Practice tab with weakness-prioritized review sessions
- Color-coded scoring: Green (exact), Yellow (close), Red (missed)
- Date input with BCE/CE toggle and range support for period events
- Mistake handling: yellow events re-shown, red questions re-asked
- Session summaries with XP calculation
- Axolotl mascot with 5 contextual moods (happy, thinking, celebrating, sad, surprised)
- Settings panel with stats overview and progress reset
- Animated XP counter, card transitions, success/failure feedback
- Timeline shows nearby events when learning new cards
- Mastery dots (Where/When/What) on timeline and practice views
- Warm antiquity design: parchment, terracotta, bronze palette
- Libre Baskerville + DM Sans typography
- Mobile-first layout (440px max-width)
- LocalStorage persistence for all progress
- Streak tracking (daily activity)
